Suponha que:
 a = 3
  b = a / 2 
  c = b + 3.1
Qual é o valor de c? 
a) c = 4.6 
b)c = 4.1 
c) c = 4 
d) Nenhuma das opções acima 
e) Não é possível determinar o valor de c 
resposta: 
Depende do valor dado a variavel 
caso1: c=int logo é letra C
caso2: b=int  c =float ou double logo é a letra B 
caso3: b=float e c=float logo é a letra A
